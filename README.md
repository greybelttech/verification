# Grey Belt Tech Document Verification

This public repository stores verification records for official Grey Belt Tech documents.

Each record includes the document ID, version, publish date, and SHA-256 hash for an official PDF. Grey Belt Tech verification pages use these records to help confirm that a document has not been modified.

This repository does not store private source files, subscriber data, or downloadable lead magnets.

Official site: https://www.greybelttech.com
